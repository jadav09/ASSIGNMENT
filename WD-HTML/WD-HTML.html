  • What are logical and physical tags in HTML?

    • In HTML the formatting tags are divided into two categories:


    (A) physical tag:-

    bold,italick,underline,smal ,big yeh sabhi tag ate hai.

    They are different from physical tags because physical tags are used to decide the appearance of the text and do not provide any information about the text.
    (yeh sare tag koi jankari prada nahi karta.)


    (B) Logical tag:-

    blockquote,del,ins,pre,quotation,strong

    these tags are used to add some logical or semantic value to the text.
    (yeh tag ki khud ki logical value hoti hai)

    Logical tags are used to tell the browser what kind of text is written inside the tags.
    (hey browser ko batata hai ki tag under kis type ka text likha gya hia )
